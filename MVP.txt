## Executive Summary

This MVP delivers a **real-time voice conversation system** where users speak naturally and receive AI-generated voice responses with minimal delay. The key innovation is **streaming sentence-level pipelining**: audio generation begins before the LLM completes its full response.

### Core Value Proposition
- **Sub-2-second response time** from user speech end to audio playback start
- **Natural conversation flow** with proper turn-taking
- **Production-ready architecture** using battle-tested components
- **Scalable design** supporting multiple concurrent connections

## Optimized Model Selection

### Decision Matrix: Model Selection by Priority

| Component | Primary Choice | Backup Option | Rationale |
|-----------|---------------|---------------|-----------|
| **VAD** | Silero VAD v5 | WebRTC VAD | Best accuracy/speed tradeoff, ONNX runtime |
| **STT** | Faster-Whisper (distil-large-v3) | Whisper v3-turbo | 5-7x faster than base Whisper, minimal accuracy loss |
| **LLM** | Gemini 1.5 Flash-8B | GPT-4o-mini / Claude Haiku | Fastest TTFT (~180ms), excellent streaming |
| **TTS** | Kokoro-82M | StyleTTS2 / F5-TTS | 82M params = 50-100ms latency, natural prosody |

#### 2. Speech-to-Text (STT)

**Selected: Faster-Whisper (distil-large-v3)**

```python

Model: distil-large-v3
Quantization: int8
Avg Latency: 180-350ms (for 3-5 sec audio)
```

**Why Faster-Whisper + Distil?**
- ✅ CTranslate2 optimization: 5-7x faster than HuggingFace Whisper
- ✅ Distil variant: 50% smaller, only 0.4% WER degradation
- ✅ Streaming-friendly: Can process in chunks if needed
- ✅ GPU/CPU flexible: Falls back gracefully

## Latency Optimization Strategy

### Latency Breakdown & Optimization

| Stage | Baseline | Optimized | Technique |
|-------|----------|-----------|-----------|
| **Network Upload** | 100ms | 50ms | • 20ms chunks<br>• Binary protocol<br>• Local testing reduces to ~5ms |
| **VAD Processing** | 500ms | 300ms | • Lower silence threshold (300ms)<br>• Skip if clear end markers |
| **STT Inference** | 600ms | 250ms | • Distil model<br>• int8 quantization<br>• beam_size=1 |
| **LLM TTFT** | 400ms | 180ms | • Gemini Flash-8B<br>• Short system prompt<br>• Low max_tokens |
| **Sentence Wait** | 800ms | 400ms | • Short responses<br>• Sentence boundary hints in prompt |
| **TTS Inference** | 300ms | 80ms | • Kokoro-82M<br>• CPU inference (avoid GPU context switch)<br>• Pre-loaded model |
| **Network Download** | 100ms | 50ms | • Streaming chunks<br>• WebSocket keep-alive |
| **TOTAL** | **2800ms** | **1310ms** | **53% reduction** |

### Critical Optimizations

#### 1. Parallel Processing Architecture

```python
async def process_utterance(audio_buffer: bytes):
    """
    Parallel pipeline with async tasks
    """
    # Step 1: STT (blocking)
    transcript = await stt_service.transcribe(audio_buffer)
    
    # Step 2: LLM Streaming (async generator)
    sentence_stream = llm_service.generate_stream(transcript)
    
    # Step 3: TTS + Send (parallel)
    tasks = []
    async for sentence in sentence_stream:
        # Launch TTS in background
        task = asyncio.create_task(
            synthesize_and_send(sentence, websocket)
        )
        tasks.append(task)
    
    # Wait for all audio to be sent
    await asyncio.gather(*tasks)

async def synthesize_and_send(text: str, ws: WebSocket):
    """
    TTS + WebSocket send in single task
    """
    audio_bytes = await tts_service.synthesize(text)
    await ws.send_bytes(audio_bytes)
```

#### 4. Audio Chunk Streaming

```javascript
// Client-side audio queue
class AudioQueue {
  constructor() {
    this.queue = [];
    this.isPlaying = false;
    this.audioContext = new AudioContext({ sampleRate: 24000 });
  }

  async enqueue(audioBytes) {
    const audioBuffer = await this.audioContext.decodeAudioData(audioBytes);
    this.queue.push(audioBuffer);
    
    if (!this.isPlaying) {
      this.playNext();
    }
  }

  playNext() {
    if (this.queue.length === 0) {
      this.isPlaying = false;
      return;
    }

    this.isPlaying = true;
    const buffer = this.queue.shift();
    const source = this.audioContext.createBufferSource();
    
    source.buffer = buffer;
    source.connect(this.audioContext.destination);
    source.onended = () => this.playNext();
    source.start();
  }
}
```

### Backend Core Implementation

```python
# main.py
from fastapi import FastAPI, WebSocket, WebSocketDisconnect
from services import VADService, STTService, LLMService, TTSService
import asyncio

app = FastAPI()

# Initialize services (singleton pattern)
vad = VADService()
stt = STTService()
llm = LLMService()
tts = TTSService()

@app.websocket("/ws/audio")
async def websocket_endpoint(websocket: WebSocket):
    await websocket.accept()
    
    audio_buffer = bytearray()
    silence_chunks = 0
    SILENCE_THRESHOLD = 15  # 15 chunks * 20ms = 300ms
    
    try:
        while True:
            # Receive audio chunk
            data = await websocket.receive_bytes()
            
            # VAD processing
            is_speech = vad.detect_speech(data)
            
            if is_speech:
                audio_buffer.extend(data)
                silence_chunks = 0
            else:
                silence_chunks += 1
            
            # Trigger pipeline on silence
            if silence_chunks >= SILENCE_THRESHOLD and len(audio_buffer) > 0:
                # Process in background to keep socket alive
                asyncio.create_task(
                    process_and_respond(audio_buffer, websocket)
                )
                audio_buffer.clear()
                silence_chunks = 0
                
    except WebSocketDisconnect:
        print("Client disconnected")

async def process_and_respond(audio: bytes, ws: WebSocket):
    """
    Main processing pipeline
    """
    # 1. STT
    transcript = await stt.transcribe(audio)
    
    # 2. LLM Stream
    sentence_buffer = ""
    async for token in llm.generate_stream(transcript):
        sentence_buffer += token
        
        # Check for sentence end
        if token.rstrip() in ['.', '!', '?']:
            # 3. TTS + Send (non-blocking)
            asyncio.create_task(
                synthesize_and_send(sentence_buffer.strip(), ws)
            )
            sentence_buffer = ""

async def synthesize_and_send(text: str, ws: WebSocket):
    """
    TTS synthesis and immediate send
    """
    audio_bytes = await tts.synthesize(text)
    await ws.send_bytes(audio_bytes)
```

### Frontend Core Implementation

```typescript
// app/page.tsx
'use client';

import { useState, useRef, useEffect } from 'react';

export default function VoiceChat() {
  const [isRecording, setIsRecording] = useState(false);
  const wsRef = useRef<WebSocket | null>(null);
  const audioContextRef = useRef<AudioContext | null>(null);
  const audioQueueRef = useRef<AudioBuffer[]>([]);
  const isPlayingRef = useRef(false);

  useEffect(() => {
    // Initialize WebSocket
    wsRef.current = new WebSocket('ws://localhost:8000/ws/audio');
    
    wsRef.current.onmessage = async (event) => {
      const audioData = await event.data.arrayBuffer();
      playAudio(audioData);
    };

    return () => wsRef.current?.close();
  }, []);

  const startRecording = async () => {
    const stream = await navigator.mediaDevices.getUserMedia({ audio: true });
    const audioContext = new AudioContext({ sampleRate: 16000 });
    audioContextRef.current = audioContext;
    
    const source = audioContext.createMediaStreamSource(stream);
    const processor = audioContext.createScriptProcessor(4096, 1, 1);
    
    processor.onaudioprocess = (e) => {
      const inputData = e.inputBuffer.getChannelData(0);
      const int16Array = new Int16Array(inputData.length);
      
      for (let i = 0; i < inputData.length; i++) {
        int16Array[i] = Math.max(-32768, Math.min(32767, inputData[i] * 32768));
      }
      
      wsRef.current?.send(int16Array.buffer);
    };
    
    source.connect(processor);
    processor.connect(audioContext.destination);
    
    setIsRecording(true);
  };

  const playAudio = async (audioData: ArrayBuffer) => {
    const audioContext = audioContextRef.current || new AudioContext();
    const audioBuffer = await audioContext.decodeAudioData(audioData);
    
    audioQueueRef.current.push(audioBuffer);
    
    if (!isPlayingRef.current) {
      playNextInQueue();
    }
  };

  const playNextInQueue = () => {
    if (audioQueueRef.current.length === 0) {
      isPlayingRef.current = false;
      return;
    }
    
    isPlayingRef.current = true;
    const buffer = audioQueueRef.current.shift()!;
    const source = audioContextRef.current!.createBufferSource();
    
    source.buffer = buffer;
    source.connect(audioContextRef.current!.destination);
    source.onended = playNextInQueue;
    source.start();
  };

  return (
    <div className="flex flex-col items-center justify-center min-h-screen">
      <button
        onClick={isRecording ? stopRecording : startRecording}
        className="px-8 py-4 text-xl font-bold rounded-full"
      >
        {isRecording ? '🔴 Recording...' : '🎤 Start Talking'}
      </button>
    </div>
  );
}
```

## Error Handling & Edge Cases

### Critical Error Scenarios

| Error Type | Handling Strategy | User Experience |
|------------|------------------|-----------------|
| **WebSocket Disconnect** | Auto-reconnect with exponential backoff | "Reconnecting..." message |
| **STT Failure** | Return generic response, log error | "I didn't catch that, could you repeat?" |
| **LLM Timeout** | Fallback to cached response or error | "I'm having trouble thinking right now" |
| **TTS Failure** | Return text response only | Show text fallback |
| **GPU OOM** | Reduce batch size, clear cache | Slight latency increase |
| **Network Congestion** | Buffer management, skip frames | Audio quality reduction notification |

### Graceful Degradation

```python
# Fallback chain for TTS
async def synthesize_with_fallback(text: str):

        return await kokoro_tts.synthesize(text)
    except Exception as e:
        logger.warning(f"Kokoro failed: {e}")
        try:
            return await styletts2.synthesize(text)
        except Exception as e2:
            logger.error(f"All TTS failed: {e2}")
            return text  # Return text as last resort
```
